# Movie app

A react app that allows the user to search for movies, see their details and add or remove them to a favourite list. 

## Development Tools

* Javasript
* Sass
* React
* Axios

### `API`

The movie data was retrieved from [OMDb API](/http://www.omdbapi.com/)
